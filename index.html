SKILLS

HTML

HTML stands for Hypertext Markup Language. It is a markup language used to create web pages and web applications. HTML uses a set of markup tags to define the structure and content of a web page.

CSS

CSS stands for Cascading Style Sheets. CSS provides a way to separate the presentation of a web page from its content, allowing designers to control the layout, colors and fonts of a web page without affecting the underlying HTML structure.

Tailwind

Tailwind CSS is a utility-first CSS framework that provides a set of pre-defined CSS classes that can be used to easily style HTML elements. Tailwind provides a set of classes that can be applied to elements to style them in an efficient way.

Bootstrap

Bootstrap is a popular open-source CSS framework that provides a set of pre-built HTML, CSS, and JavaScript components for building responsive, mobile-first web pages and web applications.

Javascript

JavaScript is a high-level programming language that is used to add interactivity and dynamic behavior to web pages.JavaScript can be used to perform a variety of tasks on web.

Vue.js

Vue.js is an open-source JavaScript framework for building user interfaces and single-page applications.Its syntax is easy to learn and understand, and it can be used incrementally.
